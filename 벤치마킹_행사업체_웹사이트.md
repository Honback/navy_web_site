## 1. 국내 상위 행사 업체 사이트 구조 분석

### 1.1 이너트립 (innertrip.co.kr) - 기업 워크샵 플랫폼

**사이트 구조 (Best Practice)**

```
[헤더]
├── 로고
├── 상품 (프로그램 / 숙박·대관 / 섭외·렌탈)
├── 견적 받아보기 (장바구니 아이콘 + 수량)
└── 로그인

[메인 페이지 섹션 순서]
1. 히어로: "준비는 가볍게, 기억은 깊게" + 대표 이미지
2. 파트너 로고: 카카오/삼성/SK/LG/네이버/현대/KB/한화 (2줄 회전)
3. 고객 후기 슬라이더: 기업별 사례 + 로고 + 평가
4. 누적 실적: 9,000회 / 350,000명 / 5,000사 (카운터 애니메이션)
5. 포트폴리오 사례: 6가지 실제 진행 사례 (썸네일+메타정보)
6. 4단계 프로세스: 견적 → 맞춤견적 → 확정 → 진행
7. CTA 배너: 상품보기
8. 포트폴리오 요청 폼: 회사명/이름/연락처/이메일
9. 푸터: 회사정보, 고객센터, 약관
```

**UX 흐름**: 신뢰 구축(로고+실적) → 사례 증명(포트폴리오) → 프로세스 명확화 → 전환(견적 폼)

**디자인 특징**:
- 반응형 이미지 (3840px 원본 → 적응형 리사이즈)
- 고객사 로고 32x32px 아이콘 + 후기 슬라이드
- 스키마 마크업으로 SEO 강화
- 전화 상담 CTA 상시 노출

## 2. 국내 행사 업체 사이트 공통 구조 패턴

### 2.1 필수 페이지 (6개)

| 순서 | 페이지 | 핵심 콘텐츠 |
|------|--------|------------|
| 1 | 홈 (랜딩) | 히어로 + 실적 + 서비스 요약 + CTA |
| 2 | 서비스/프로그램 | 카테고리별 서비스 상세 |
| 3 | 포트폴리오/갤러리 | 진행 사례 사진/영상 |
| 4 | 회사소개 | 연혁, 팀, 비전, 고객사 |
| 5 | 견적/문의 | 견적 요청 폼 또는 상담 신청 |
| 6 | 블로그/자료실 | 행사 팁, 사례 스토리 |

### 2.2 메인 페이지 섹션 순서 (검증된 패턴)

```
1. 히어로 섹션 ──── 슬로건 + 대표 비주얼 + 메인 CTA
2. 신뢰 지표 ────── 고객사 로고 배열 (회전/정적)
3. 서비스 소개 ──── 카테고리 카드 (3~6개)
4. 누적 실적 ────── 숫자 카운터 (건수/참여자/고객사)
5. 포트폴리오 ──── 사례 카드 (이미지+타이틀+태그)
6. 고객 후기 ────── 슬라이더 (기업로고+담당자코멘트)
7. 프로세스 ────── 단계별 진행 과정 (3~5단계)
8. CTA 배너 ────── 견적 요청 / 무료 상담
9. 푸터 ──────── 회사정보 + SNS + 약관
```

### 2.3 전환 요소 (CTA)

| 위치 | CTA 유형 | 예시 |
|------|---------|------|
| 헤더 (고정) | 전화 상담 | "070-xxxx-xxxx" 버튼 |
| 히어로 | 메인 CTA | "견적 받아보기" / "무료 상담" |
| 서비스 카드 | 개별 CTA | "자세히 보기" / "프로그램 보기" |
| 중간 배너 | 유도 CTA | "포트폴리오 받기" / "상품 보기" |
| 하단 폼 | 리드 수집 | 이름/회사/연락처/이메일 입력 |
| 플로팅 | 상시 노출 | 카카오톡 채널 / 전화 아이콘 |

## 9. 핵심 인사이트 요약

### 반드시 갖춰야 할 것 (Must-Have)
1. **풀스크린 비디오/이미지 히어로** - 행사 현장의 생동감 전달
2. **고객사 로고 배열** - 신뢰도 즉시 확보
3. **포트폴리오 갤러리** - 유형별 필터링 가능
4. **견적 요청 폼/전화 CTA** - 전환 경로 항상 가시적
5. **모바일 반응형** - 담당자의 50%+ 모바일 검색

### 차별화 요소 (Nice-to-Have)
1. **장바구니형 견적 시스템** (잔치꾼 벤치마킹)
2. **실시간 알림톡** 진행상황 공유
3. **숫자 카운터 애니메이션** (실적 강조)
4. **인스타그램 피드 실시간 삽입**
5. **블로그/인사이트** SEO 콘텐츠

### 피해야 할 것 (Anti-Pattern)
1. 구형 디자인 (Flash 시대 레이아웃)
2. 가격 불투명 (전화로만 확인)
3. SNS 부재 / 업데이트 안 된 사이트
4. 모바일 미대응
5. 포트폴리오 없이 서비스 설명만 나열
