## 4. 강점

### 1) 즉시 견적 확인 시스템
- 홈페이지에서 제품 가격을 즉시 조회 가능
- 장바구니 기반 견적 카트 시스템으로 원하는 물품/수량을 담으면 바로 견적 확인
- 행사 업계 최초 **실시간 가격 투명 공개** 모델

### 2) '바로 견적' + '비교 견적' 이중 서비스
- **바로 견적**: 물품과 수량 선택 시 즉시 금액 확인
- **비교 견적**: 여러 업체 견적을 한번에 비교 가능
- 행사 담당자의 견적 비교 시간을 대폭 단축

### 3) 알림톡 기반 실시간 진행 상황 관리
- 카카오 알림톡으로 행사 진행 상황 실시간 확인
- 행사 담당자의 불안감 해소

### 4) 원스톱 서비스
- 견적 → 제작 → 소통까지 하나의 플랫폼에서 해결
- 물품 렌탈부터 인력 섭외, 케이터링까지 통합 제공
- 무료 기초 상담 제공

### 5) 행사 유형별 패키지
- 개관식 세트, 시상식 세트 등 유형별 패키지로 초보 담당자도 쉽게 준비
- "무엇이 필요한지 모르겠다"는 담당자를 위한 가이드 역할

## 7. 종합 평가

**포지셔닝**: 행사 물품 렌탈 및 기획의 **"쿠팡형 이커머스 플랫폼"**

잔치꾼은 기존 행사 업계의 불투명한 가격 체계와 복잡한 견적 과정을 디지털화하여, 행사 담당자가 온라인 쇼핑하듯 행사를 준비할 수 있게 만든 플랫폼입니다. 특히 행사 경험이 적은 담당자(공공기관, 중소기업)에게 유용하며, 유형별 세트 서비스로 "무엇이 필요한지 모르겠다"는 고민을 해결해줍니다.

**적합한 고객**: 행사 물품 렌탈이 주 목적인 기관/기업, 행사 기획 경험이 적은 담당자, 빠른 견적 비교가 필요한 경우
